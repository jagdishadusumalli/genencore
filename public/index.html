SERVICES & SKILLS



Web Development :

We have extensive experience in building high quality web 2.0 applications using opensource technologies primarily Ruby On Rails (RoR). We have built many scalable web solutions for clients across the globe. From basic social integrations to big data business solutions and to building API’s for services we have done it all.

Technologies and Tools we employ : Ruby on Rails, Postgres, Mysql, Linux, Ubuntu, Nginx, Apache Server, Monit.


Web Design :

Semantic front end engineering is our strength. We have been building solutions which work seamlessly irrespective of device types and operating systems. Responsive web design is second nature to us

Technologies and Tools we have employed : HTML, CSS, Javascript, Coffescript, HAML, SASS, SCSS, Yeoman, Grunt, Node.js, AngularJS, Bootstrap and Foundation frameworks


Mobile Apps Development :

We believe some apps has to be living ON the device.
So we have the skills and good experience on building high performance cross platform native applications for Android and IOS Platforms

Cloud Solutions :
